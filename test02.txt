1.首先進到git

2.輸入這兩行設定基本資料
  git config --global user.name "s15113135"
  git config --global user.email "s15113135@stu.edu.tw"

3.輸入位置
  git clone https://github.com/s15113135/test.git

4.在資料夾中建立一個你的程式
  ex:test02.txt (記事本)

5.追蹤你的程式
  git add test02.txt

6.查看現在git狀態
  git status

  你會發現以下：
  On branch master
  Your branch is up-to-date with 'origin/master'.
  //這裡會有一些被追蹤的程式(顯示紅色)，如果沒有東西代表沒有程式被追蹤
  nothing to commit, working directory clean

7.追蹤到了以後，我們可以用註解來幫助其他開發者或是自己知道，什麼時候更改了這個程式
  git commit

8.這時會進到了編輯器裡頭

  編輯器有以下幾個簡單指令：
  i 進入編輯模式(自由打字啦)
  Esc 退出編輯模式(回到一般模式)
  dd 刪除一行(一般模式下)
  :wq 存檔離開(一般模式下)
  :q! 不存檔直接離開 (一般模式下)

9.此時按下i後可以輸入註解

10.輸入完畢按下esc離開編輯

11.輸入 :wq 可以存檔離開

12.上傳程式
   git push
