1.首先進到git 

2.輸入這兩行設定基本資料
  git config --global user.name "s15113135"
  git config --global user.email "s15113135@stu.edu.tw"

3.接著初始化
  git init 

4.複製別人的資料存放處
  git clone https://github.com/s15113135/test.git

5.在目錄中建立一個你的程式
  ex:test02.txt (記事本)

6.進入到你的目錄
  cd test //cd 目錄名稱

7.追蹤你的程式
  git add test02.txt

8.查看現在git狀態
  git status

  你會發現以下：
  On branch master
  Your branch is up-to-date with 'origin/master'.
  //這裡會有一些被追蹤的程式(顯示紅色)，如果沒有東西代表沒有程式被追蹤
  nothing to commit, working directory clean

9.追蹤到了以後，我們可以用註解來幫助其他開發者或是自己知道，什麼時候更改了這個程式
  git commit

10.這時會進到了編輯器裡頭

  編輯器有以下幾個簡單指令：
  i 進入編輯模式(自由打字啦)
  Esc 退出編輯模式(回到一般模式)
  dd 刪除一行(一般模式下)
  :wq 存檔離開(一般模式下)
  :q! 不存檔直接離開 (一般模式下)

11.此時按下i後可以輸入註解

12.輸入完畢按下esc離開編輯

13.輸入 :wq 可以存檔離開

14.如果覺得進到 commit 很麻煩，也可以利用 -m 來快速提交
   git commit -m "輸入你要的註解內容"

15.上傳程式
   git push 

   首次 git push 會要你輸入你的 github 帳號密碼 輸入完後即可上傳
